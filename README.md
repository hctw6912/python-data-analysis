Python 데이터 분석 및 시각화 프로젝트


이 저장소는 인프런 강의 "나도 코딩 데이터분석 시각화"를 학습하며 진행한 Python 데이터 분석 및 시각화 실습 프로젝트입니다.

강의 정보
강의명: 나도 코딩 데이터분석 시각화

강의 링크: https://www.inflearn.com/course/나도코딩-데이터분석-시각화

강의 내용: Python을 활용한 데이터 수집, 전처리, 분석, 시각화 기초부터 실습까지 다룹니다.

프로젝트 개요
Python과 Pandas, Matplotlib 등 주요 데이터 분석 라이브러리를 활용해 다양한 데이터셋을 분석하고 시각화합니다.

수집한 데이터 전처리, 그룹화, 필터링, 시계열 처리 등을 실습하며 데이터 분석 역량을 강화합니다.

시각화를 통해 데이터의 인사이트를 효과적으로 표현합니다.
